# Core API Structure & Middleware

## Establish the foundational API structure with essential middleware for security, logging, and error handling.

### Acceptance Criteria

- Express.js server with TypeScript configuration
- Security middleware implemented (helmet, cors, rate limiting)
- Comprehensive logging system with Winston
- Global error handling middleware
- Request validation middleware with Joi/Zod
- Health check endpoints implemented

### Technical Requirements

- Express.js with TypeScript decorators
- Security packages: helmet, cors, express-rate-limit
- Winston logger with multiple transports
- Joi or Zod for request validation
- Custom error classes and handlers
